# 🎓 Universal Academic Forecaster (v0.6)

A cute, pastel 8-bit aesthetic degree forecaster and grade calculator built for UK university students. Easily calculate your final classification for your BSc, MSc, or track your PhD milestones. 

## ✨ Features
* **🎨 Cozy 8-Bit Design:** Fully styled with a pastel pixel-art aesthetic, warm typography, and interactive background sprites.
* **🏫 Live University Search:** Automatically fetches and caches every recognized institution in the UK using the Hipo Labs University API.
* **💾 Auto-Save:** Powered by `localStorage`. Your grades, module names, and progress are saved locally as you type, so you never lose your data.
* **🎵 Ambient Sound Engine:** Uses the built-in Web Audio API to generate synthetic retro sounds on interaction (clicks, sliders, winning sounds, and hidden Easter eggs).
* **🖨️ Clean Export:** Print or save your final forecast as a PDF with dedicated print stylesheets that hide the messy UI elements.
* **📈 Dynamic Grading:** Swap between Undergrad, Master's, and Doctorate grading structures with ease.

## 🚀 How to Run
This is a fully self-contained Single Page Application (SPA). There are no dependencies, frameworks to compile, or databases to setup.

1. Clone or download this repository.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Select your university and degree type, and start tracking!

## 🛠️ Built With
* **HTML5 & Vanilla JS:** Core logic and state management.
* **Tailwind CSS (via CDN):** Rapid structural styling.
* **Custom CSS:** Themed pixel-art animations, shadows, and UI control styling.
* **Canvas Confetti:** For that sweet First-Class / Distinction celebration.
* **Web Audio API:** For synthesizing sound effects without needing external `.mp3` files.
